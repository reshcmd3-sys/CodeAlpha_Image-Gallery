# Image Gallery

A responsive and visually appealing Image Gallery developed using HTML, CSS, and JavaScript. This project allows users to view and explore a collection of images in an organized and interactive layout.

## 📸 Features

- Responsive image gallery layout
- Clean and modern user interface
- Hover effects on images
- Smooth user experience
- Optimized for desktop and mobile devices
- Easy navigation through image collections

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

image-gallery/
│
├── index.html
├── style.css
├── script.js


## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Ensure all images are stored in the images folder.
4. Open `index.html` in a web browser.
5. Browse and enjoy the gallery.
